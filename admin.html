Заявки с формы

Дополнительные получатели — Telegram user ID через запятую
Заявки с формы сайта рассылает Telegram-бот. Каждый получатель должен один раз написать боту /start, иначе Telegram не даст боту прислать сообщение. Свой user ID можно узнать у бота @userinfobot. Токен бота задаётся не здесь, а в настройках хостинга: Railway → Variables → TG_BOT_TOKEN. Основной получатель заявок зашит в код сервера. Чтобы новые ID заработали для всех посетителей — вкладка «Экспорт» и обновление data.js, как с остальным контентом.
